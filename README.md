# Dreams

Strategic Inventory Management & Retail Solutions for Modern Enterprises.

## Project info
High-fidelity stock synchronization, secure audit logging, and automated retail workflows.

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Technologies Used
- **Core**: Vite, TypeScript, React
- **UI & Styling**: shadcn/ui, Tailwind CSS, Framer Motion
- **Database & Auth**: Supabase
- **Icons**: Lucide React

## Key Features
- Real-time Inventory Tracking
- Multi-Warehouse Support
- Secure Audit Logs
- Automated Point of Sale (POS)
- Staff & Role Management
